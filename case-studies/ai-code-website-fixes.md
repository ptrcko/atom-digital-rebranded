## Overview

Applied AI-assisted diagnostics and fixes to resolve website stability and performance issues.

## Business problem

- Customer-facing site was experiencing outages and degraded performance.
- Root causes were unclear and undocumented.
- Recovery needed to be rapid without introducing new risk.

## Approach

- Used AI-assisted tooling to surface code anomalies and configuration issues.
- Prioritised fixes based on customer impact and security considerations.
- Documented changes and monitoring to prevent recurrence.

## Outcome

- Restored site stability and improved response times.
- Clear runbooks and monitoring implemented for ongoing operations.
- Stakeholders gained confidence through transparent updates and metrics.
