# Fractional CTO / COO Website

This repository contains the source code for the personal website of a Fractional CTO / COO and Technology Partner.  
The site is designed to clearly communicate the service offering, demonstrate credibility through case studies, and highlight partnerships with key platforms and tools.

The goal is to position 20 years of C-level technical and operational experience into a modern, service-driven consultancy site that appeals to:

- Non-technical startups needing a fractional CTO  
- Traditional businesses needing a fractional COO  
- Companies seeking AI rollout support  
- Organisations needing project-based technology guidance  
- Businesses looking for operational, automation, or systems improvement
